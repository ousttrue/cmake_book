# Variables

`${PROJECT_NAME}` のように `$` を先頭に置いて波括弧で変数名を囲む。

| name                   |                                                                          |
|------------------------|--------------------------------------------------------------------------|
| PROJECT_NAME           | project(name) が変数に入る。                                             |
| CMAKE_CURRENT_LIST_DIR | CMakeLists.txt のあるディレクトリー                                      |
| CMAKE_SOURCE_DIR       |                                                                          |
| CMAKE_BINARY_DIR       | CMakeの作業ディレクトリー。{doc}`/commandline` {doc}`/cmakelists/output` |
